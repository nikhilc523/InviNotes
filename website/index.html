Everything you need

Invisible to Screen Share

Uses OS-level setContentProtection to hide from Zoom, Meet, Teams, and OBS.

Real-time Collaboration

Live cursors, presence indicators, and instant sync powered by Yjs CRDT.

Rich Text Editor

Notion-style slash commands, headings, code blocks with syntax highlighting, task lists.

Deep Link Sharing

Share invinotes://join/... links that open the app and join a room instantly.

Always on Top

Stays above all windows. Adjust opacity or enable click-through to interact with apps behind it.

Global Hotkeys

Show/hide, move, resize, snap, and adjust opacity — all from the keyboard.

How It Works

1

Create a Room

Click Share → Create new room to start a collaboration session.

2

Share the Link

Copy the invite link or room ID and send it to your collaborator.

3

Collaborate Invisibly

Edit together in real time. Your notes are invisible to screen capture.
